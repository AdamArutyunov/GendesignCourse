Рисуем красивое с помощью кода

Когда-то дизайн рисовали вручную чернилами, потом печатали на станке, ещё позже появились графические редакторы и айпад.

Сегодня дизайн переходит в эпоху автоматизации. Люди описывают творческие идеи с помощью кода, а роботы берут на себя оставшуюся рутинную работу. Автоматизация позволяет дизайнеру тратить время не на получение конкретного экземпляра решения, а на описание принципов, по которым скрипт создаст сразу множество вариативных решений.

Мы верим, что программная генерация — самый перспективный способ делать принципиально новый дизайн. На этом курсе мы научим вас основам программирования и покажем самые популярные приёмы, которые используют генеративные художники.

Что будет

Вы научитесь генерировать картинки, которые можно использовать в своих дизайнах, как только фантазии хватит. Для этого изучим:

  • Основы языка программирования JavaScript с нуля — расскажем только тот минимум, который нужен для практики;
  • Рисование с помощью библиотеки p5.js;
  • Пять классических алгоритмов генеративного арта.

Чего не будет

  • Вёрстки, HTML и CSS;
  • Основ дизайна;
  • Нейросетей.

Программа

  • Четыре теоретических занятия

    1. Язык программирования JavaScript. Отладка и консоль. Типы данных и арифметические операции. Переменные.

    2. Ветвления и логические операции. Циклы. Область видимости переменной.

    3. Массивы. Функции, параметры и аргументы.

    4. Что такое p5.js. Как сохранять результат работы. Документация.

  • Сетка
  • Сабдивижн
  • Клеточные автоматы
  • Работа с изображениями
  • Флоу-филд
